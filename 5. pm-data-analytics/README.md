# 데이터 분석 (pm-data-analytics)

PM을 위한 데이터 분석 스킬: SQL 쿼리 생성 및 코호트 분석. 사용자 데이터 분석, 쿼리 생성 및 유지율(Retention) 패턴 파악을 지원합니다.

## 스킬 (3)

- **ab-test-analysis** — 통계적 유의성, 샘플 크기 검증, 신뢰 구간 및 실행 가능한 권장 사항을 포함하여 A/B 테스트 결과를 분석합니다.
- **cohort-analysis** — 사용자 참여 데이터에 대해 코호트 분석을 수행합니다.
- **sql-queries** — 자연어 설명을 바탕으로 SQL 쿼리를 생성합니다.
